>	5.6 编写程序，交换某个整数的奇数位和偶数位，使用指令越少越好（也就是说，位0与位1交换，位2与位3交换，依次类推）。

相邻奇偶位的4种情况：

```
00 ^ 00 = 00
01 ^ 11 = 10
10 ^ 11 = 01
11 ^ 00 = 11
```

```java
public int CCI5_6(int n) {
	int m = n;
    int k = 3;//11B
    int mask = 0;

    while(m != 0) {
    	//相邻奇偶位为01或10，对应位为11
    	if((m & 1) != ((m >> 1) & 1))
        	mask = mask | k;

        //从右至左确定相邻奇偶位的对应位
        k = k << 2;
        m = m >>2;
    }

    return n ^ mask;
}
```
下面的思路可以简化指令：

将奇数位右移一位，将偶数位左移一位，再将两次移动的结果合并

```java
return ((n & 0x55555555) << 1) | ((n & 0xaaaaaaaa) >> 1);
```
